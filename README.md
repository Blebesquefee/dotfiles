# Dotfiles Repository
This repository is used to store all my configurations files using [chezmoi](https://www.chezmoi.io/)
program for my [Asahi Linux](https://asahilinux.org/) system.

## This config's repository contains :
- **Window Manager** : [i3](https://github.com/i3/i3)
- **Terminal** : [terminator](https://github.com/gnome-terminator/terminator) 
- **Shell** :
    [zsh](https://github.com/z-shell)
/    [oh-my-zsh](https://github.com/ohmyzsh/ohmyzsh)
/    [powerlevel10k](https://github.com/romkatv/powerlevel10k)
/    [starship](https://starship.rs/)
- **Lock Screen** : [betterlockscreen](https://github.com/betterlockscreen/betterlockscreen)
- **Fancy Config** :
    [neofetch](https://github.com/dylanaraps/neofetch)
/    [picom](https://github.com/dylanaraps/neofetch)
/    [polybar](https://github.com/adi1090x/polybar-themes)
/    [rofi](https://github.com/adi1090x/rofi)
/    [dwall](https://github.com/adi1090x/dynamic-wallpaper)
- **Text Editor** : [vim](https://github.com/vim/vim)
- Other configuration's scripts

## Not included in this repository:
- **Login Manager** : [sddm](https://github.com/sddm/sddm)
